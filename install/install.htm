使用企业建站系统协议

(一) 运行环境需求: PHP+MYSQL
建议PHP版本在4.1以上
MYSQL版本在4.0以上
系统大量采用ajax技术建议服务器APACHE在2.0以上

(二) 安装步骤:
( 如果您在安装过程中出现问题或有疑问,请进入 http://www.oi77.com/bbs/thread.php?fid-19.html 提问)

(1) Linux 或 Freebsd 服务器下安装方法 第一步: 使用ftp工具中的二进制模式,将该软件包里的 upload 目录及其文件上传到您的空间 假设上传后目录仍旧为 upload 第二步: 先确认以下目录或文件属性为 (777) 可写模式 data data/sql_config.php data/bbscache data/groupdb data/style attachment htm_data image image/upload image/cn_img 第三步: 运行 http://yourwebsite/upload/install.php 安装程序,填入服务器配置信息与创始人 相关资料, 完成安装!

(2) Windows 服务器下安装方法 第一步: 将解压后的文件上传至你的空间，保持目录结构不变，假设目录为upload 第二步: 运行 http://yourwebsite/upload/install.php 安装程序,填入服务器配置信息与创始人 相关资料, 完成安装!

(三) 系统使用者未经过本人同意不得更改系统版权

(四) 房地产系统是开源化系统提供下载学习，您需要使用于商业请与我们取得联系。没有购买系统不提供功能升级服务，只提供错误和不安全性升级。
